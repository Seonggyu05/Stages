# Stages
## "Structuring your workflows."

  Stage Manager Tools

  A collection of utility applications designed to enhance and automate window management and stage switching for Stage
  Manager on iPadOS and macOS.

  📦 Included Applications

  This repository contains the latest pre-compiled versions of the following tools:

  1. StageOrganizer
  StageOrganizer is a management utility that helps you coordinate multiple application windows and "Stages." It
  provides the logic for grouping related tasks and ensuring your workspace remains organized during complex workflows.

  2. StageMap
  StageMap acts as a visual bridge and transition handler. It is primarily used to facilitate smooth movement between
  different stages and provides the system with the necessary signals to handle "handover" events between application
  clusters.

  ---

  🚀 Getting Started

  Installation
   1. Download the latest .zip file from this repository. Unzip the file. 
   2. macOS: Move the apps to your /Applications folder.
   3. iPadOS: Install via your preferred deployment method (e.g., Apple Configurator, TestFlight, or AltStore).

  URL Schemes
  These apps utilize custom URL schemes for automation and inter-app communication:
   - stageorganizer://
   - stagemap://

  You can use these schemes in Shortcuts or other automation tools to trigger stage transitions programmatically.
